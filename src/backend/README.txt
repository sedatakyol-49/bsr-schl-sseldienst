IONOS PHP mail setup

1. Upload the generated site contents to your IONOS webspace.
2. Open contact-config.php in the web root.
3. Replace BITTE-HIER-IHR-SMTP-PASSWORT-EINTRAGEN with the real mailbox SMTP password.
4. Keep contact.php, contact-config.php and the phpmailer folder in the same web root.
5. Test the contact form on https://bsr-schluesseldienst.de/kontakt

Recommended SMTP defaults for IONOS:
- Host: smtp.ionos.de
- Port: 587
- Security: tls
- Username: Info@bsr-schluesseldienst.de
